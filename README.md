# CarShare — каршеринг (MVP)

Курсовой проект: веб-приложение каршеринга на основе ТЗ (UC-01 — UC-07).
Запускается локально, без внешних БД и без реальных платежей/SMS.

## Запуск
npm install
npm start

Откройте http://localhost:3000

## Что внутри

- **Регистрация** с SMS-кодом (код показывается во всплывающем «push-уведомлении» вместо реальной SMS, также пишется в консоль).
- **Загрузка ВУ и паспорта** с имитацией OCR.
- **Карта** (Leaflet + OpenStreetMap) с автомобилями вокруг пользователя.
- **Бронирование** на 15 минут с холдом депозита.
- **Поездка** с чек-листом осмотра, таймером и расчётом стоимости.
- **Привязка карты** без реального списания.
- **История поездок** в профиле.

## Сброс данных

Удалите `data/db.json` и содержимое `public/uploads/` — при следующем запуске парк авто пересоздастся.
